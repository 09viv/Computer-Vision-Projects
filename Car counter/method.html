Car Counter

This project utilizes several computer vision techniques to accurately detect and count cars in a video feed.

1. Background Subtraction

The system uses background subtraction to isolate moving objects (vehicles) from the static background. The createBackgroundSubtractorMOG2 method from OpenCV is employed, which is effective in detecting shadows and variations in the background.

2. Contour Detection

Once the moving objects are isolated, contours are detected using the findContours function. This function identifies the boundaries of the detected vehicles.

3. Vehicle Counting

The system maintains a list of detected vehicles and counts how many cross a predefined counting line. This is done by checking the position of the center points of the detected vehicles against the counting line.

4. HTML Reporting

The current vehicle count is saved in an HTML file each time a vehicle crosses the counting line, providing a simple way to review the results.

5. Libraries Used

  • OpenCV: For image processing and computer vision tasks.
  • NumPy: For numerical operations and array manipulations.

6. Real-Time Processing

The system processes video frames in real-time, providing immediate feedback on vehicle counts as they pass through the detection line.
